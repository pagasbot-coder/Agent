## Baseline

Стартовые скоры: **62** по всем доменам (янтарь), затем дельты, clamp **15…95**.

## Сигналы → домены

| Сигнал | Домен | Дельта (ориентир) |
|--------|-------|-------------------|
| Стороны: 0 строк | D1 | −25 |
| Стороны: 1–2 | D1 | −12 |
| Стороны: ≥4 с ролями | D1 | +5 |
| У стороны «молчание» содержит «Высок» | D1, D8 | −8 / −5 (cap) |
| Риск `открыт` + влияние В | D5, D8, D4 | −8 / −10 / −5 |
| Каждый открытый риск | D8 | −4 (cap −24) |
| «Не знаем» с непустым вопросом | D8 | −7 (cap −28) |
| Бюджет: delta начинается с `+` | D6, D7 | −6 / −5 |
| Веха без статуса / «просроч» | D4, D6 | −6 / −5 |
| Этап 0–1 | D3 +4, D4 −8 | ранняя зрелость плана |
| Этап 4–5 + мало открытых рисков | D5, D6 | +5 / +4 |
| Этап 4–5 + много открытых В-рисков | D5, D6 | −10 / −8 |
| Решения ≥3 «принято» | D3 | +6 |

## Принципы

1. Не выдавать «зелёный сертификат» — при дырах в реестрах оставлять жёлтый/красный.  
2. Пустые реестры ≠ здоровый проект: сильнее штрафуем D1/D8.  
3. Человек на радаре может поправить любой score после apply.
