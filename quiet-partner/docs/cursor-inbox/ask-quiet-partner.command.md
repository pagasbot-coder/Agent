---
description: Ответить по снимку Тихого напарника (cursor-inbox)
---

Прочитай `@quiet-partner/docs/cursor-inbox/latest.md` (если нет — попроси пользователя скачать снимок из Тихого напарника: радар → «Спросить в Cursor» → «Скачать снимок MD» и сохранить как `quiet-partner/docs/cursor-inbox/latest.md`).

Затем:
1. Ответь на «Вопрос пользователя» из файла по-русски, коротко.
2. Опирайся на оценки доменов и выжимку пульта; не выдумывай факты вне снимка.
3. При нехватке данных — 1–3 уточняющих вопроса.
4. Предложи один фокус на сегодня, если уместно.
5. Напомни: co-pilot, не сертификация PMBOK.
