## Подготовка маркетинга (7 шагов) — до старта работ

Документы: [не знаем](reestry/05-ne-znaem.md) · [матрица](shablony/02-matricza-otvetstvennosti.md) · [план связи](shablony/02-plan-svyazi.md) · [риски](reestry/02-riski.md) · [бюджет](reestry/03-byudzhet.md) · [вехи](reestry/06-vekhi.md) · [изменения границ](reestry/07-izmeneniya-granic.md)

1. **Установочная встреча команды** — правила, роли, темп.
2. **Знаем / Не знаем** — каждый «не знаем» → исследование или запрос клиенту ([реестр](reestry/05-ne-znaem.md)).
3. **Структура** — что делаем, кто нужен, сколько единиц; **одно** главное KPI успеха.
4. **Матрица ответственности** — один A на результат ([шаблон](shablony/02-matricza-otvetstvennosti.md)).
5. **Сроки / вехи** — + double-check со специалистами + слот на комментарии клиента ([вехи](reestry/06-vekhi.md), [план связи](shablony/02-plan-svyazi.md)).
6. **Риски + деньги** — резерв 10–20 % ([риски](reestry/02-riski.md), [бюджет](reestry/03-byudzhet.md)).
7. **Связь со стороной** — кто решает; ритм созвона + [еженедельный статус](shablony/04-ezhenedelnyy-status.md).

**Рубеж:** клиент и команда смотрят один план; критичные «не знаем» закрыты.

## А — Агентство (реклама, соцсети, контент)

| Фаза | Фокус | KPI (выбери 1–3 главных) |
|------|-------|---------------------------|
| Подготовка | Аудитория, оффер, доступы, цепочка креатива | — |
| Неделя | Срез + гипотезы | стоимость заявки/покупки, окупаемость, вовлечённость, рост |
| Контроль | Медиабюджет + согласование креатива | расход против плана |
| Передача | Кабинеты, отчёт, усиливать / остановить | |

**Риски:** кабинет не пополнен; модерация; клиент тянет согласование; перегруз на 3 проектах.  
**Вопрос:** какой один KPI главный — не размыт ли десятком?

## Б — Брендинг

**Развилка в старте**

- Продукт на полке / массовый сервис → чаще **продуктовый** → правда из аудитории и статистики.  
- Штучный B2B / мало клиентов / вместе с заказчиком → чаще **корпоративный** → правда из интервью и топов.

| Этап | Артефакт у заказчика | Риск |
|------|----------------------|------|
| Исследование | Отчёт с **выводами**, не свалка фактов | Нет доступа к людям |
| Позиционирование | Ценности, тон, метафора | Коллективный заказчик тянет в стороны |
| Имя / визуал | Короткий список, направления | Владелец ≠ аудитория |
| Стандарты | Брендбук под **их** носители + оглавление ссылок | Красивый PDF без внедрения |

**Вопрос:** откуда правда — полка или кабинеты директоров?

## В — Маркетинг внутри компании

| Принцип | Практика |
|---------|----------|
| Организатор | Собери мастерство вокруг измеримого сдвига поведения |
| Ядро 6–10 | Кто даёт 80 % успеха — отношения сейчас |
| Бриф со смыслом | Зачем / для кого / какой сдвиг + свобода исполнителю |
| Поле | Пользуйся продуктом; говори с пользователем |
| Обратная связь | Точность > громкость; о работе, не о личности |
| Разбор | Результаты (совершенный вид), не только процессы |

**Вопрос:** задача меняет поведение — или это суета?
